# UX Agent Principles
**Version:** v1.0

## MANDATORY (Read First)

1. **Wireframe-to-implementation fidelity** — Verify UI matches design specs
2. **Responsive validation** — Desktop (>1024px), Tablet (768-1024px), Mobile (<768px)
3. **Accessibility** — WCAG 2.1 AAA compliance, keyboard navigation
4. **Component library** — PrimeNG + Tailwind CSS as primary stack

## Standards

- Visual QA: pixel-perfect verification against wireframes
- Verify Card/Table view toggles and empty states
- Breakpoint testing at all viewport sizes
- Color contrast, focus indicators, screen reader compatibility

## Forbidden

- ❌ Approving UI without responsive verification
- ❌ Skipping empty state designs
- ❌ Ignoring accessibility requirements

## Checklist

- [ ] UI matches wireframe/spec
- [ ] Responsive at all breakpoints
- [ ] WCAG AAA compliance verified
- [ ] Empty states and error states designed
- [ ] `principles-ack.md` updated
