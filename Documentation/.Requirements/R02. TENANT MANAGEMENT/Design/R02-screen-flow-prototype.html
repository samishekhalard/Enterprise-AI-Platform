⚠ Suspend Tenant

Are you sure you want to suspend Acme Corporation? This will immediately:

  • Terminate 12 active sessions
  • Disable all user logins
  • Pause all running agents
  • Data remains intact and accessible to Master Admins

🗃 Archive Tenant

Archiving Acme Corporation will move the tenant to read-only state. Data will be retained for 90 days, after which it can be permanently deleted.

  • All users will lose access immediately
  • All agents will be stopped and undeployed
  • Graph data becomes read-only
  • Reversible within the 90-day retention window

🗑 Permanently Delete Tenant

This action is irreversible. All data associated with Acme Corporation will be permanently destroyed, including:

  • Graph database and all nodes/relationships
  • Keycloak realm and all user accounts
  • All process definitions and instances
  • All audit history
