# Diagram Repository

Use this folder to store TOGAF/ArchiMate diagram source and exported views.

## Suggested Diagram Set

- Business capability map.
- Value stream overview.
- Application cooperation view.
- Data dissemination/ownership view.
- Technology platform decomposition.
- Transition architecture views.

## Source Format Guidance

- Preferred model source: ArchiMate modeling tool export.
- Documentation-friendly supplemental format: Mermaid embedded in markdown.
- Keep each diagram linked from its owning phase document.
