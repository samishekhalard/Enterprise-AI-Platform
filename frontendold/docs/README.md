## Technology Stack

- **Framework:** Angular 21.x (standalone components)
- **State Management:** NgRx Signals
- **UI Components:** Angular Material + Custom Design System
- **Testing:** Jasmine + Karma (unit), Cypress (E2E)

## Quick Start

```bash
cd frontend
npm install
ng serve
```

Access at: http://localhost:4200
